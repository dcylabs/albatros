Docker is widely used to build, ship and run applications or services.
This docker image is (and will remain) an open-source container that provide a secure WebUI that helps you to manage your Docker daemon from everywhere.
It also comes with a free android application that is more user friendly for those who wants to use it through their smartphones.
For Apple users, there is no mobile application planned at the moment because I do not have the hardware to work on it.
